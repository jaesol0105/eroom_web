# Eroom_Web
<b>http://eroom.kro.kr</b>

스마트 독서실 예약 웹 페이지<br>

## Features
> 사용자는 독서실의 좌석을 선택하여 예약 할 수 있다. <br>
> 실시간으로 좌석 배치도와, 이용 가능한 / 이용 중인 좌석이 표시된다. <br>

* 로그인
* 회원가입
* 좌석 예약
  * 좌석을 예약 합니다.
  * 배치도와 이용 가능한 / 이용 중인 좌석을 출력합니다.
* 좌석 이용 종료
  * 현재 이용 중인 좌석의 이용을 종료합니다.
* 마이 페이지
  * 회원 정보 수정
  * 회원 탈퇴
